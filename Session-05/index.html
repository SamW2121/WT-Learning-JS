Setting Up

Download the file session-05.zip from teams.

Unzip the contents and place into your "Learning JS" project.

Open the Laragon terminal.

Use cd %userprofile% to get to your home folder (cd ~ on Mac).

Use cd to change into the folder with your Learning JS project (something like: cd Source/Repos/Learning-JS/

run npm install @tailwindcss/typography

run npm install @tailwindcss/forms

run cd Session-05

now run npx tailwindcss -i ./src/tailwind-main.css -o ./assets/css/tailwind.css --watch

or see Session-03 for how to set up.
