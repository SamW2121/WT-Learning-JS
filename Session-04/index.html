Three columns

Create a 3 column layout in Tailwind using the following Emmet code (remember this is on ONE line):

div.grid.grid-cols-3.gap-4.my-2>
    (section.border.bg-stone-400.border-zinc-400.rounded-md.p-4>
        p{Hello $}
    )*3

Remember to press TAB at the end of the line.

This creates a 3 column grid, with each section having a mid-stone background, a zinc border and padding inside the rectangle.
