Setting Up

  • Download the starter files from Teams
  • Uncompress the Session-06.7z file
  • Open WebStorm (or whichever editor you prefer)
  • Move the Session-06 folder into your Learning JS project
  • open laragon
  • run the laragon terminal
  • Use these commands, altering to suit your folder structure/project location:
  • cd %userprofile%
  • cd Source/Repos
  • cd WT-Learning-JS
  • cd Session-06
  • npm install

Remember that you need to be in the Session-06 folder and then run npx tailwindcss -i ./src/tailwind-main.css -o ./assets/css/tailwind.css --watch to see the page update with any classes being added to the pages.
