Setting Up

1) Installing TailwindCSS

Open the terminal, then click on the V to open the terminal options, select the "Command Prompt" option.

This should put you in the folder with your and use the following commands:


npm install tailwindcss
npm install @tailwindcss/forms
npm install @tailwindcss/line-clamp
npm install @tailwindcss/typography
npx tailwind init

2 Configuring TailwindCSS

Create a new folder called src and then the folders assets, assets/css, assets/js, assets...

Open the new tailwind.config.js file

Edit the file to read:


/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{html,js}",
    "./**/*.{html,js}"
  ],
  theme: {
    extend: {},
  },
  plugins: [
    require('@tailwindcss/typography'),
    require('@tailwindcss/line-clamp'),
    require('@tailwindcss/forms'),
  ],
}

3 Adding the tailwind-main.css file

Add the tailwind-main.css file to the src folder

Open the file and add


/** Filename: ./Session-xx/src/tailwind-main.css **/
@tailwind base;
@tailwind components;
@tailwind utilities;

3 Running tailwind

Open the terminal and use the following command:


npx tailwindcss -i ./src/tailwind-main.css -o ./assets/css/tailwind.css --watch
If you get an error, then you will need to open the Laragon terminal and navigate to your project's folder, before then changing into the Session-03 folder.

4 Previewing

Hover over the HTML file and then click on the browser you wish to preview in.

Your page will be rendered.

The page will re-render when you save your files.

It may not pick up all changes immediately due to time to detect and then update the CSS file automatically.

5 Adding Classes

Tailwind Has many classes to use. For example:

text-blue-50 text-red-100 text-amber-500 text-stone-900 rounded border border-blue-500 mx-auto m-6 p-2 py-6 gap-4

and many more...
