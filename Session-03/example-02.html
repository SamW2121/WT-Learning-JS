If you have not read the ReadMe.md file then you will need to run the following command for the Tailwind Form utilities to be available:

npm install -D @tailwindcss/forms

Then add the require line to the plugins section of your tailwind.config.js, as shown below:

plugins: [
    require('@tailwindcss/forms'),
    // ... other plugins added here, after the forms plugin
  ],
