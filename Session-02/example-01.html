Setting up for Tailwind CSS (For remaining examples)

This is based on the instructions on https://tailwindcss.com/docs/installation

Modify the WT-Learning-JS "template" folder

Open your WT-Learning-JS project.

Expand the Session-xx folder

Create a new folder called src in the Session-xx folder

In this folder create a new css file tailwind-main.css.

Add the @tailwind directives for each of Tailwind’s layers to your tailwind-main.css CSS file.

@tailwind base;
@tailwind components;
@tailwind utilities;

Install Tailwind CSS

Install tailwindcss via npm, and create your tailwind.config.js file. 

npm install -D tailwindcss
npx tailwindcss init

Configure your template paths

Add the paths to all of your template files in your tailwind.config.js file.

/** @type {import('tailwindcss').Config} */
module.exports = {
    content: [
        "./src/**/*.{html,js}",
        "./**/*.{html,js}"
    ],
    theme: {
        extend: {},
    },
    plugins: [],
}

Start the Tailwind CLI build process

Run the CLI tool to scan your template files for classes and build your CSS.

npx tailwindcss -i ./src/tailwind-main.css -o ./assets/css/tailwind.css --watch

Start using Tailwind in your HTML

Add your compiled CSS file to the head and start using Tailwind’s utility classes to style your content.

<link href="/assets/css/tailwind.css" rel="stylesheet">

This file uses the tailwind installation that the steps above explain.
